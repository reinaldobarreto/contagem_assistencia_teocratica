# Guia de Uso

## Tela Contagem
- Preencha ocupação, selecione indicadores e adicione notas.
- Salve para registrar a reunião.

## Tela Registros
- Use a barra de busca para filtrar por texto (inclui indicadores).
- Aplique filtros de período e tipo de reunião conforme necessário.
- Escolha a ordenação desejada.
- Menu Exportar:
  - Exportar CSV (filtrados): gera texto CSV e abre compartilhamento.
  - Exportar PNG (filtrados): captura a lista atual como imagem e compartilha.

## Exportar para revisão
- Envie o CSV para análise ou o PNG para visualização rápida.
